and
Moonisland (moonisland.itch.io)

Controls:
WASD - walk
Shift - run
Space - jump
Arrow keys - look around
E - interact
ESC - back
Enter - confirm
F2 - save screenshot
F11 - toggle fullscreen

Game uses a lot of textures from soviettextures.com, please visit that awesome website!

This game uses these Java libs:
Lightweight Java Game Library
Java OpenGL Math Library
Luaj
